<include>

Defines a .CFM template to <cfinclude>, creating HTML that will be viewable by the user.  

 

Model-Glue will search the mappings listed in the ViewMappings setting (defined in the application's ColdSpring.xml file) in order, searching for the template defined in the TEMPLATE attribute.

Attributes

  1. Name (Required) - The name by which the rendered content will be known in the ViewCollection

  2. Template (Required) - The filename of a .CFM template inside one of the directories in the ViewMappings setting

  3. Append (Optional, default = "false")  - If an include by this NAME has already been rendered, should the content from this include be appended to the existing content or should it overwrite the existing content?  (False will overwrite, True will append.)

Child Tags

  1. <value>
